<h2><a href=https://www.codewars.com/kata/52b757663a95b11b3d00062d/train/go target="_blank">WeIrD StRiNg CaSe</a></h2><h3>6 kyu</h3><p>Write a function that accepts a string, and returns the same string with all even indexed characters in each word upper cased, and all odd indexed characters in each word lower cased. The indexing just explained is zero based, so the zeroth index is even, therefore that character should be upper cased. <em>Indexing resets for each word</em>. In other words, the first letter of every word is index <code>0</code> (even), so it must always be uppercase, etc.</p><p>The passed in string will only consist of alphabetical characters and spaces(<code>' '</code>). Spaces will only be present if there are multiple words. Words will be separated by a single space(<code>' '</code>).</p><h3 id="examples">Examples:</h3><pre><code>"String" =&gt; "StRiNg""Weird string case" =&gt; "WeIrD StRiNg CaSe"</code></pre>
