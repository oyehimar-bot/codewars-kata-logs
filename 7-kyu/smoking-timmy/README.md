<h2><a href=https://www.codewars.com/kata/5a0aae48ba2a14cfa600016d/train/python target="_blank">Smoking Timmy</a></h2><h3>7 kyu</h3><p>Timothy (age: 16) really likes to smoke. Unfortunately, he is too young to buy his own cigarettes and that's why he has to be extremely efficient in smoking.</p><p>It's now your task to create a function that calculates how many cigarettes Timothy can smoke out of the given amounts of <code>bars</code> and <code>boxes</code>:</p><ul><li>a bar has 10 boxes of cigarettes,</li><li>a box has 18 cigarettes,</li><li>out of 5 stubs (cigarettes ends) Timothy is able to roll a new one,</li><li>of course the self made cigarettes also have an end which can be used to create a new one...</li></ul><p>Please note that Timothy never starts smoking cigarettes that aren't "full size" so the amount of smoked cigarettes is always an integer.</p>
